# Bmarket
##### A web based application for exchanging, selling and buying used books.
##### Application is available here: [156.17.130.129:8080]()
#### Technology stack:
* Spring
* Thymeleaf
* Javascript
* CSS bootstrap
* MySQL
* Hibernate
* Tomcat
* Git
* Jenkins
